# MonGenEditor

适配 LF 引擎的刷怪批量修改计算器

## 项目简介

**MonGenEditor** 是一个专为 LF 游戏引擎定制的刷怪批量参数编辑与计算工具。该工具帮助开发者和关卡设计师更加高效地批量生成、管理和修改刷怪参数，提高关卡编辑效率，节省重复性工作时间。

## 功能特性

- 批量读取、编辑和生成用于 LF 引擎的刷怪参数文件
- 支持常见参数的自动计算与批量应用
- 结果可导出为 LF 引擎可直接使用的格式
- 操作简便，降低配置复杂度

## 安装方法

1. 克隆本仓库：

    ```bash
    git clone https://github.com/3dchaos/MonGenEditor.git
    ```

2. 进入项目目录并安装依赖（如有 `requirements.txt`，请补充）：

    ```bash
    cd MonGenEditor
    pip install -r requirements.txt
    ```
